# BOQONNAA 7 – Amantoonni Hundinuu Wangeelaa Labsuu fi Tajaajiluu ni Danda'u

Yesus Kiristoos kan du’e namoota hundaafi. Namni hunduu isa waliin walitti dhuufeenya addaan hin cinne akka godhu ni barbaada. Kun oduu baay’ee nama gammachiisu waan ta’eef nama hundaaf labsuu qabna. Yesus kirstoos Matewoos 28:18-20 irratti gara biyya lafaa hunda adeemnee duuka bu’oota akka horannuuf nutti himee jira. Kun yeroo baay’ee “ergama isaa guddaa” jedhamuun beekkama. Wangeela labsuu fi tajaajiluun carraawwan gammachuu nuuf uuman qofaa oss hin ta’in, ajaja Gooftaa keenyaa Yesus kirstoosi. Modeelonni lamaan armaan gaditti argaman waldoonni Kiristiyaanaa tokko tokko tajaajila keessatti eenyu akka hirmaachisan kan itti tarreeffamaniidha.


Modeela – 1




Moodeela kana keessatti tajaajilli kan qindaa’u fi hojii irra kan oolu dursitoota waldaatiini. Adeemsa kana keessatti, tajaajilli kun haala sirrii ta’een kana gochuuf namoota ga’umsa qabaniin waan hojjetamuuf yeroo tokko tokko bu’a qabeessa ta’ee ni argama. Modeela kan keessatti miseensonni waldaa gaggeessitoota isaanii kan gargaaran yoota’u; tajaajilawwan waldaadhan alatti jiran kamiyyuu irratti hin hirmaatan. Haata’u malee boqonnoota darban muraasa keessatti akkuma ilaalle immoo Islaamummaan dhiibbaa geessisaa kan jiru bifoota hawaasa hundarratti dha. Qaamni Kiristoos deebii Islaamummaadhaf kennituun bu’a qabeettii ta’uu kan dandeechuu bifoota hawaasichaa hundarratti hirmachuu yoo dandeechedha.


Waldoota modeela kanan fayyadaman keessa miiseensonni jiran tajaajiltoota waldaa waan hin taanef taajaajila keessatti kallattiidhan hirmaattota hin akka ta’an itti hin dhagahamu.




Modeela tajaajilaa kana keessatti immoo dursitoonni waldaa miseensota tajaajilaan ni deeggaru. Dursitoonni waldaa miseensotaa tajaajilawwan ijaarsa waldaatin ala jiran irratti akka bobba’an ni hidhachiisu; ni jajjabeessu. Dursitoonni kutaalee tajaajilaa kanneen irratti hirmaattota ta’anis miseensonni isaanii kaka’umsa mataa isaanitiin carraalee tajaajilaaf ta’an barbaaduu irraa akka of qusatan hin godhan. Cufummayyuu miseensonni; waaldaa keessas ta’ee ijaarsa waldaatin ala tajaajila jiru irratti hamma danda’an hunda akka hirmaatan ni jajjabeessu. Waldaanis namoonni Wangeela waliin gahuu, duuka bu’uummaa akkasumas tajaajila Musliimotaaf godhamu dabalatee kutaa tajaajilaa hundaaf eddoo itti leenji’an nitaatii.


Amantoonni eddoowwan itti jiraataniif itti hojjetan gaggeessitoota waldaa caala beekun isaanii modeela tajaajilaa kana faayida qabeessaa isa taasisa. Dabalataanis namoota lakkoofsi isaanii baay’ee ta’e tajaajila irratti akka hirmaatan isaan taasisa. Kunis waldaan fuulaawwan hawaasaa hundaa irratti dhiibbaa gaarii, kan bal’aa ta’ee fi bu’a qabeessaa ta’ee akka geessistu ishee gargaara. Haala kanaan Kiristiyaanonni biyyattii keessatti argaman marti isaanii Wangeela tamsaasuu, haqadabsummaa balaaleffachuun akkasumas Islaamummaan jiraattota biyyattii irratti mormii mirgaa fi mormii walabummaa fiduuf deebii laachudhaan itti gaafatama isaanii ni bahu.


Roomaa 12:3-8 Qaamni tokko hojiiwwan garagaraa kan qaban kutaalee gara garaa akkuma qabu hunda, nuti kiristiyaanonni waldaa fi hawaasa keenya keessatti gahee fi hojii gara garaa akka qabnu nu barsiisa. Tokko tokko kan waamamaniif arjooman akka kennaniif ta’a.Warri biroo immoo kan waamamaniif dursitoota, jajjabeessitoota, barsiistota yookin bulchitoota akka ta’aniifidha. Kennaa bifa kamiyyuu yoo qabaattan rakkina hin qabu. Wanti guddaan kennaawwan Kunneen ulfina Waaqa fi tajaajila isaatif oolchuf danda’uu keessani. Kanaafuu Kiristiyaanni hunduuIslaamummaadhaf deebii kennuf akka danda’u wanta tokko gochuu ni danda’a jechuudha .



Wangeela Tamsaasuu

Musliimota wangeelan qaqqaabuun qamni Kirisitos martuu Islaamummaadhaf deebii kan ittiin kennu karaa isa ol’aanadha. Macaafni Qulqulluun tamsaasa wangeelaa irratti hirmaachuu akka dandeenyu qofa osoo hinta’in dirqama nuuf kenname akka ta’es ifaan nidubbata.



Kadhannaadhan Jalqabuu

Musliimotaaf dhugaa ba’uu osoo hin jalqabin dura hafuuran kan qophoofne ta’uu qabna. Kan jalqabnus kadhannaadhan ta’uu qaba. Gooftaan nun hundaaf humnaa fi angoo ittiin dhugaa baanu waan nuuf kenneef (Mat 28:18-20) galanni isaaf haata’u. Gara Musliimota onneen isaanii banaa ta’ee fi waa’ee Kiristoos dhagahuuf qophaa’ota gara warra ta’anii akka isin qajeelchu akkasumas tokkoon keessan sodaan yoo isin keessa jiraateyyuu kaka’umsa akka isiniif kennu kadhadhaa. Dhiiga Kiristoosin karaa hundaan akka eegumsa isiniif kennusisa gaafadhaa. Gooftaan misiraachoo isa gaarii Musliimotaafakka geessuu dandeechan akka isin gargaaru fi jaalala inni Musliimotaaf qabu akka isiniif kennu akkasumas yeroo dubbattan ogummaan akka isiniif ta’ukadhadhaa. Musliimonni baay’en abjuudha fi mul’ataan itti mul’atee yookin Macaafa Qulqulluu dubbifatanii akkasumas dhuga ba’umsa Kiristiyaanotaa dhagahanii gara Kiristoos akka dhufan yaadachuu dhan ofii isaa akka isaanitti mul’isu isa kadhadha.



## Kan Qoodnu Maali?

Musliimota waliin ennaa dubbattan Macaafa Qulqulluu, jaalala Yesus Kiristoos fi cubbuun keenya akka dhiifamuuf fannoo irratti akka nuuf du’e itti himuu irratti xiyyeeffadhaa. Kuni Islaamummaa qeequu irra filatamaadha. Musliimotaaf Allaah baay’ee fagoo fi dhuunfaa isaanitti isa waliin walitti dhuufeenya qabaachuu waaqa hin dandeenyedha. Yesus Kiristoos jireenya keessan keessatti maal akka hojjetee fi jireenya keessan keessatti maal hojechaa akka jiru Musliimotatti itti himaa.


Yeroo baay’ee mariiwwan waaq-qo’annaa (theology) walxaxoon faayida qabeeyyii ta’anii hin argaman. Wangeela Musliimotaaf beeksisuudhaf seenaawwan Macaafa Qulqulluu kaasudhaan seenaawwan Kanneen fayyadamuun waa’ee eenyummaa Waaqayyoo isaanif ibsitanii dabalatanis jireenya keessan keessatti jijjiirama akkamii akka fidan itti himuun karaa bu’a qabeessa. Fakkenyaf:- seenan mucaa isa bade jaalalli Waaqayyoo haalota irratti kan hin hundoofna akka ta’ee fi jaalala isaa hojii keenyan akka nuti argachuu hin dandeenye agarsiisuu danda’a.seenan cuphaa Yesusimmoo waa’ee Sadantokkoo (Trinity) ibsuudhaf nu gargaara. Seenawwan Yesus warra dhukkubsatan itti fayyisee fi Seexana itti baase, humnaa fi angoo isaa mul’isu. Du’aa ka’uun Yesus humna  inni du’arratti qabu agrsiisa. Seenaawwan kanneen Musliimotatti himuun isaan waliin michuummaa akka uumnus nu gargaara.


Haalota akka waa’ee fayyaa fi waa’ee maallaqa jireenya Musliimotaa keessa darban irratti kadhachuudhaf akka fedha qabdan itti mul’isaa. Akka kadhattaniif hammam fedha akka isaan qaban yemmuu agartan isin ajaa’iba. Haata’u malee“Hin barbaadu” yoo isiniin jedhan akka garaa akka isin hin hanqanne. Musliimni tokko akka Kiristiyaanni isaaf kadhatu hayyame jechuun tarkaanfii dheeraa adeemera jechuudha. Kadhannaa keessan dhaabbataa keessatti baay’istanii akka isaanif kadhaattan itti himuu ni dandeessu. Kunis Musliimotaaf akka isin yaaddan itti ni mul’isa. Waaqayyo kadhannaa keessan akka dhaga’uu fi jireenya isaanii keessatti giddu galummaa waaqummaa akka raawwatu amanaa.


Karan biraa kan itti isin Musliimonni waa’ee Gooftaa Yesus akka beekan gochuu dandeessan immooMacaafa Wangeelaa yookinMacaafaQulquluu bifa kennaatin isaanif kennuunidha. Dubbisa isaanii Matewoos yookin Luqaas irraa yoo jalqaban gaarii ta’a.



Namoota Nuti hin Beeknef Dhugaa Ba’uu

Wangeela namoota hundaaf akka labsinu waamamnee jirra. Itti yaadnee ba’uudhan Musliimotaaf wangeela labsinuuf fedha qabaachuu ni dandeenyaa yookin karaa nuti ittiin hin eegnen Waaqayyo gara isaanii nu qajeelchuu ni danda’a.Musliimotaaf dhugaa ba’uuf adeemuu keessannin dura kadhannaaddhan yeroo murtaa’e dabarsuu keessan mirkaneeffadhaa. Yeroo dhugaa baatanis karaa nagahaatin haata’u. Bifa jaalalaan guutameen Wangeela dhugaa ba’uu yemmuu yaaltan yoo isin fudhachuu baatan hin dinqamina; hin gufatinaas; Wangeela dhaga’uudhaf onneen isaanii banaa kan ta’e Musliimonni heddu ni jiruuti.


Warraa Jijjiraman Kunuunsuu

Musliimni tokko Kiristiyaana ta’uudhaf erga murteessee booda waldaadhan hordoffiin godhamuunif barbaachisadha. Kiristiyaanonni haarofni baay’een mana isaanii, maatii isaanii, hojii isaani fi madda galii isaanii dhabuu ni danda’u. Gamtaan Kiristiyaanaa Ugaandaa keessatti argamu tokko Kiristiyaanota haarawa akkamitti kunuunsuu akka qabnu haala gaariidhaan nu barsiisa. Kiristiyaanonni haarofti duuka bu’ummaa qofa osoo hin ta’in waan akka uffata hodhuu , hojii sibiilaa, konkolaataa suphuu, hojjii bareedinaa leenjiiwwan fakkaatan nikennamuuf. Mallaqa jireenyaaf isaan barbaachisu akka horataniifi mataa isaanii akka of danda’an meeshaaleen daldala xinnoo ittiin eegalan ni kennamuuf.


Musliimotaaf ennaa dhugaa banutti dhiironnii fi dubartoonni Musliimaa ija gara garaatin akka ilaallamanii fi kopha kophaa akka taa’an qalbeeffaachuu qabna. Aadaa Islaamummaa keessatti maatii gidduuti yoo ta’e malee dhiironni dhiira dubartoonni dubartii qofa waliin dubbatu. Kiristiyaanni saala faalla ta’e tokko Musliimota dubbisuuf yoo yaale isaan dallansiisuu danda’a.


Musliima akka tasaa argattan, fakkeenyaf konkolaataa keessatti kan isin cinaa taa’ef akka dhugaa baatan Hafuurri Qulqulluun isin kakaasuu ni dandaa’a. Yeroo kanattis dursitanii qophaa’uu dhabaattan illee jecha dubbachuu qabdan isa sirri ta’e Waaqayyoo wanta isiniif qopheessuf sodaachuu hin qabdan.



Michuuwwan Keenyaf Dhugaa Ba’uu

Michummaa uumun Musliimataaf Yesus Kiristoos beeksisuuf karaa filatamaadhaa. Musliimonni akkuma keenya namootadha. Gammachuu, gaddaa fi dhiphaachuun akkuma nu qunnamu, isaanis ni qunnama. Akka namummaa isaanitti akkataa jireenya isaanifi aadaa isaanii baruuf yaalii taasisaa. Akkasumas waa’ee jireenya keessanii isaan akka beekan godhaa. Kiristiyaana ta’uu jechuun maal jechuu akka ta’ee fi hariiroo Yesus Kiristoos waliin isin qabdan isaan taphachiisaa.


Yeroo Musliimota waliin taatan haala sirriidhan uffachuu hin dagatinaa. Dabalees saala faallaa waliin yeroo hin fudhatinaa. Keessumattuu harka keessaniinuu yoo ta’e haala barammadhan akka michoota keessan biro waliin taasistan isaan hin tuqinaa, taphaaf qaama isaanii hin dhahinaa yookin akka harka keessaniin hin hammanne of eeggadhaa. Dhiironni Kiristiyaanaa michummaan isaanii dhiirota musliimaa waliin yoo ta’u, Michummaan dubartoota Kiristiyaanaa immoo dubartoota Musliimaa  waliin ta’uu qaba. Dubartoonni Kiristiyaanaa dhiirota Musliimaa waliin kophaa isaanii haalli isaan yeroo ittiin dabarsan akka hin jiraanne of eeggachuu qabu.



Kiristoos Fudhachuu

Musliimonni isin dhugaa baataniif Kiristoosin akka Gooftaa fi Fayyisaa isaanitti fudhatanii isa hordofuu qabu. yeroo sirrii kana itti gochuu dandeessan akka hafuurri Waaqayyoo isinitti mu’isu isa gaafadhaa. “Waqayyo jireenya keessan keessatti baruu ni barbaadduu?” yookin “waa’ee GooftaaYesus beekuuu bira darbitanii isa dhuunfaa keessanitti beekuu hin feetanii?” yookin “yeroo duutan gara motummaa Waaqayyoo deemu keessaniif mirkana argachuu hin feetanii?” jechuu dhan jalqabuu ni dandeessu.


Gaafilee kanneen Musliimota gaafachuun sirritti akka yaadan gochuu fi Yesus gara jireenya isaanii akka affeeraniif isaan gargaaruu ni dandeessu. Gara murtiitti dhufuuf yeroo waan isaan barbaachisuu danda’uuf obsuu qabdu.Yesus Kiristoos fudhatan jechuun Musliima ta’uu isaanii dhaaban jechuu akka ta’e jabeessitanii itti himuus qabdu.



Duuka Bu’aa Gochuu fi Hordofuu

Kiristiyaanota akkuma ta’uu keenya namni tokko gara Gooftaa dhufe jechuun isaaf maal jechuu akka ta’e hubachuu qabna. Islaamummaa dhiisun akka gantummatti lakkaawamee maatii, firoota yookin abbootii angoo irraa adabbii itti fiduu ni danda’a. Kunis hamaa fi sukkaneessaa ta’uu ni danda’a. Yeroo tokko tokko Kiristiyaanonni haarofti miidhan irra ga’uu, reebamuu fi hojii isaanii irra fi manaa ari’amuu ni danda’u. Dhirsoota yookin niitolee isaani dhaabuu fi ijoollen isaanii irra fudhatamuun yoo hammaate immoo ajjeefamuu ni danda’u. Kiristiyaanonni haarofti eddo itti Musliimonni isaan arguu danda’anitti amantaa isaanii dhabuu fi ijoollen isaanii irraa fudhatamuun, yoo hammaate immoo ajjeefamuu ni danda’u. Kiristiyaanonni haarofti eddoo itti Musliimonni isaan arganitti amantaa isaanii yoo labsan yookan yoo cuuphaman balaan uumamuu ni danda’a. Seeronni amantaa kunneen raawwatamuu kan qaban eddoo obboloonni isaani hordoftoonni Kiristoos qofti itti argaman eddoo foo’amaatti ta’uu barbaachisa.


Kiristiyaanonni haarofni tokko tokko lubbuu isaanitiif sodaachuu irraa kan ka’e dhokachuun irra jiraachuu ni danda’a. Waldaan naannichatti argamtu Kiristiyaanicha haarawa sanaaf maatii ta’uutu ishee irraa eegama. Waldaan eddoo boqonnaa gaarii, nyaata, michooma, walitti dhuffeenya, yoo barbaachisaa ta’es hojii kennuun ishee  barbaachisa. Keessumattuu Kiristiyaanonni haarofti Kunneen akka obboleewwaniitti simannaan “baga dhuftanii” isaanif taasifamuu qaba. Waldaan duuka bu’oota  isaan gochuuf yeroo bal’aa fudhachuu fi hordoftootaKiristoos ta’auu jechuun maal jechuu akka ta’e barsiisuu qabdi. Kiristiyaanummaan Islaamummaa irraa baay’ee kan addaa ta’e ta’uuisaa irraa kan ka’e adeemsi duuka bu’aa gochuu yeroo baay’ee fudhachuu ni danda’a. Kiristiyaanonni haarofti Kunneen garee qo’annaa Macaafa Qulquluu yookin kan biroo, fedhiin Kiristiyaanota haarawa kanneenii itti dhag’amuu kan danda’uu isaanif kan yaadufi isaanif kan kadhatu keessatti hammatamuun kan isaan barbaachisuuf kanaafidha.


Roomaa 10:10-15 fi Hojii Ergamootaa 1:8 dubbifadhaa.

Roomaan 10 fayyuudhaf maal gochuu akka qabnu nutty hima. Yesus Gooftaa akka ta’e onnee keenya keessatti amanuu fi afaan keenyaan ragaa bu’uu qabna. Fayyinni nama hundaafu hayyamamee jira. Ergaan Roomaa namni maqaa Gooftaa waamu hundi akka fayyu nutti hima. Itti fufees “namoonni (wangeela) yoo hin dhageenye ta’e akka mitti fayyu?” jechuudhan nu gaafata. Rooman 10 misiraachoo isa gaarii namoota hundaaf labsu akka dandeenytu kan nu gosru affeerri ajaa’ibsiisa dha. Kunis Musliima, Hinduu yookin amantaa aadaa gara garaa kan hordofan maatii keenya, olloota keenyaa akkasumas eddoo hojii yookin mana barumsaatti kan argannu michuuwwan keenya darbees karaa utuma deemnuu namoota argannu hunda of keessatti hammata. Yesus namni martinuu akka Gooftaa fi Fayyisaatti akka isa beekan barbaada. Ajajni kun Kiristiyaanota tokko tokkotti ulfaachuu akka danda’uu fi dhugaa ba’uudhaf akka sodaachuu dandeenyu Gooftaan ni beeka. Hojii ergamootaa 1:8 irratti Hafuurri Qulqulluun haala kana keessatti akka nu faana ta’uu fi waa’ee isaa namoota birootif humna ittiin dubbannu akka nuuf kennu nutti dubbata. Hafuurri Qulqulluun hojiiwwan dinqisisoo ta’an kan akka warra dhukkubsatan fayyisuu fakkaatan akka hojjennu nu dandeechisa. Hafuurri Qulqulluun ogummaa fi adda baasummaa akka nu keessa kaa’u kadhachuu ni dandeenyaa.
