# Dhuga-ba'umsa Caaffata Qulqulluu

Kiristiyaanummaan Kitaaba Qulqulluufi seena-qabeessa ta'e Waaqayyo qaama sadiin kan jiraatu ta'uusaa barsiisa. Barumsi kun guutummaa barsiisa Kitaaba Qulqulluu waa'ee Waaqayyoo dubbatu kan argisiisu yoo ta'u, hayyoonni Waldaa Kiristiyaana jecha «Sillaasee» kan Waaqummaatiin ibsaniiru. Sadan tokkummaa Waaqayyoo barsiisawwan hangafa Kitaaba Qulqulluu keessaa tokko ta'us "dhuga baatota Joovaa" kan fakkaatan gareewwan sobdootaafi Waaqayyo qaaman tokko ta'uusaa kan barsiisan amantaa Islaamummaa kan fakkaatan biratti mormii guddaa irratti kaasa. Gareewwan sobdoota barsiisni isaanii Kitaaba Qulqulluun faallessu yerooma hundumaa Hafuurri Qulqulluun humna eenyummaa hin haaaune qofa ta'uusaa kan barsiisan yoo ta'u Islaamummaan ammoo eenyummaa Hafuura Qulqulluu ergamticha Gabreel waliin fakkeessa. Garuu Hafuurri Qulqulluun qaamota sadan Sillaasee keessaa tokkoofi Yaahiwween Waaqayyo ta'uusaa Caaffanni Qulqulluu ifaa fi haala falmisiisaa hin taanen ragaa ba'a.

Barreeffama kanatti Hafuurri Qulqulluun Yaahiwwee Waaqayyoo ta'uusaa kan mirkaneessan barreeffamoota Caaffata Qulqulluu dhiyeessina. Dubbifattoonnis dhuga-ba'umsa Caaffata Qulqulluu akka ta'uu malutti hubachuun khasalmaa Hafuura Qulqulluu ilaalchise ijjennoo Kitaaba Qulqulluufi kan sirrii ta'e akka fudhataniif isaan jajjabeessina.

Gooftaan keenya Yesus Kristos Yaahiwwee Waaqayyoo ta'uusaa kan ittiin mirkaneessine barreeffama keenya keessatti akkuma ibsine Maqaan Waaqayyoo "Yaahiwee" jedhamu afaan Ibruu ta'uusaatiin barreeffama Kakuu Moofaa keessatti malee Kakuu Haaraatti hin argamu. Barreeffama keenya kana keessattis haala walfakkaatan barreeffamooni waa'ee Hafuura Qulqulluu Kitaaba Qulqulluu keessatti barreeffaman barreeffamoota Kakuu Moofaa keessatti barreeffaman waliin ilaaluun ragaawwan kan dhiyeessinu ta'a.

"Kottaa in saganaa gadis in jennaa, Waaqayyo uumaa keenya durattis in jilbeenfannaa! Innoo Waaqayyo keenya, nuyi saba harka isaa jala jirruu dha, hoolota inni tiksuu dhas har'a sagalee isaa utuu dhaggeeffattanii attam gaariidha. Akkuma Mariibaa iddoo morkii sanatti, akka gaafa Maasaa iddoo qoramaa isa lafa onaa keessaatti isin mata-jabeeyyii hin ta'inaa" (Far. 95:6-8).

Faaataan mootichi Daawit Seera Ba'uu 17;2 irraa hangafti Lubootaa Museen saba Israa'eliin "Waaqayyoonis maaliif qortu?" jechuudhaan waa'ee Yaahiwwee kutaa dubbate keessatti duubatti deebi'ee yaadate kan faarfate yoo ta'u, faarfatichi Yaahiwwee jedhee kan waame Waaqa Israa'el, Hafuura Qulqulluu ta'uusaa barreessaan kitaaba Ibruu akkanaan nuuf mirkaneessa:

"Kanaafis akkuma hafuurri qulluluun jedhu, "Har'a, sagalee Waaqayyoo yeroo dhageessan, akkuma warra guyyaa qoramsaa sanatti lafa onaatti, Waaqayyo irratti ka'an sanaa, isin mata-jabeeyyii hin ta'inaa" (Ibr. 3:7-9).

Bara Kakuu Moofaa Israa'elonni lafa onaa keessatti Waaqayyon qoran Yaahiwwee Waaqayyoo inni jedhamu Kakuu Haaraatti Hafuura Qulqulluu ta'uusaa ibsameera. Kunis Hafuurri Qulqulluun Yaahiwwee Waaqayyo ta'uusaa nuu mirkaneessa.

"Yeroo sanatti isaan, "Saaraan haati manaa kee eessa jirti?" jedhanii isa gaafatan; innis, "Achi dunkaana keessa jirti." ree?

Kutaa kanatti akkuma barreeffame Abrahaamiifi Saaraaf ilma (Yisihaaq) kan kenne hundumaa kan danda'u Yaahiwee yoo ta'u innis Hafuura Qulqulluu akka ture ergamtichi Phaawulos ergaa isaa Galaatiyaa keessatti akkanaan nuuf mirkaneessa:

"Ammas yaa obboloota ko, akkuma Yisihaq, isinis ijoollee abdii Waaqayyoo ti. Haa ta'u iyyuu malee, yeroo sana inni akka fedha foonii duwwaatti dhalate, isa akka fedha hafuura Waaqayyootti dhalate akkuma ari'ate, har'as immoo akkasuma in ta'a" (Gal. 4:28-29).

Kutaa kanatti Ergamtichi Phaawulos Abraham fedha foon isaatiin garbittii Aggaar irraa kan godhate Ismaa'el fedha Hafuura Qulqulluun (Yaahiween) Saaraa ishee birmaduu taaterraa kan dhalate Yisihaaqin akkuma ari'ate bara kanattis akka fedha fooniitti warreen dhalatan nuyi Hafuura Qulqulluutti lammata dhalaneen ari'atu jechaa jirra. Kanaanis Abraamiifi Saaraaf Yisihaaqin kan laate Waaqayyo Yahiwween, hundumaa kan danda'u Hafuura Qulqulluu ta'uusaa nu hubaachiisa.

"Kun dubbii Waaqayyoo isa bara Uziyan, Yotaam, Ahaaz, Hisqiyaasis tarreetti biyya Yihudaa irratti mo'anii, Yerobi'aam ilmi Yeho'aash immoo biyya Israa'el irratti mo'etti, gara Hose'aa ilma Beriit dhufee dha" (Hos. 1:1-2).

Waaqayyo Yaahiween sagalee isaa karaa raajotaan akka erguufi kana malees karaa isaaniin akka dubbatu Kitaaba Qulqulluu keessatti irra deddeebi'amee ibsamee argina. Gara Kakuu Haaraa gaafa dhufnu ergamtichi Phexros raajota kan ergu Waaqayyo (abbaa) ta'uusaafi raajotasaa geggeessuun karaa isaaniin kan dubbatu ammoo Hafuura Qulqulluu ta'uusaa akkanaan ragaa nuuf ba'a:

"Raajii dubbachuun takkaa jaalala namaatiin hin taane; garuu namoonni hafuura qulluluudhaan geggeeffamanii, wanta Waaqayyo biraa fudhatan dubbataniiru" (2Phex. 1:21).

Dubbisa kanarraa akkuma hubannu Raajota kan ergu "Waaqayyo abbaa Yaahiwee" yoo ta'u karaa Raajotaan kan dubbatu ammoo "Yaahiwee Hafuura Qulqulluudha." Kakuu Moofaa keessatti Raajota hangafaa jalqabee hanga Raajota xixiqqaa jirinatti kan argaman Caaffanni Qulqullaa'aan hundi karaa Raajotaan dubbataa kan ture Waaqayyo Yaahiween, Hafuura Qulqulluu ta'uusaa nuu mirkaneessu. (Raajotaa keessaa hangafa kan ta'ee karaa Musee dubbachaa kan ture Haahiwwee Hafuura Qulqulluu ta'uusaa kan dhugaa ba'an barreeffamoota walbira qabuun ilaala. Kes. 12:2:16:40,27, 23, Iya. 20:2-3, 1Mot. 8:53).

"Yommus ani, "Wayyoo kaa! Ani nama arrabni isaa xuraa'e, namoota arrabni isaanii xuraa'e gidduus kanan jiraadhu waanan ta'eef, iji koos Waaqayyoon mootii maccaa waan argeef nan bade!" jedhee nan iyyee...Kanaaf inni, "Gara saba kanaa dhakiitii, 'Dhaga'uu in dhageessa, garuu in hubattu; ilaaluu in ilaalta, garuu hin argitu!' Jedhiin! Itti fufees immoo, akka hin arginetti, gurra isaatiin akka hin dhageenyetti, yaada garaa isaatiinis akka hin hubannetti, gara koottis akka hin deebinetti, akka hin fayyifamnettis, yaada garaa saba kanaa hadoodchi, gurra isaa cuqqaali, ija isaas dunuufachiisi!" Jedhe" (Isa. 6:5-10).

Kutaa kanatti Raajicha Isaayasiin kan erge Waaqayyo Gooftaan (Adonay Yaahiwee) ofii isaatiin Hafuura Qulqulluu ta'uusaa Hojjetaan Wangeelaa Luqaas kitaaba Hojii Ergamootaa keessatti ragaa nuuf ba'a:

"...Phaawulos dubbii tokkicha kana isaanitti dubbate; "Hafuurri qulluluun afaan Isaayaas raajichaan abboota keessanitti, 'Gara saba kanaa daqi jedhiinis, dhaga'uu in dhageessu, garuu hin hubattan; ilaaluu in laaltu, garuu hin argitan, Yaadni garaa saba kanaa hubachuu dadhabera; gurra isaanii cuqqaalaniiru, ija isaaniis dunuunfachiisaniiru. Silaa'oo, ija isaaniitiin in argu, gurra isaaniitiinis in dhaga'u turan; yaada garaa isaaniitiin in hubatu, gara koottis in deebi'u turan; ani immoo isaan nan fayyisan ture' yommuu jedhee ba'eessa dubbate. Egaa fayyisuun Waaqayyoo kun, warra saba isaa hin ta'iniif akka darbe beekaa; isaan immoo in dhaga'u" jedhe" (HoE. 28:25-28).

Kakuu Moofaa keessatti Isaayas raajii akka dubbatuuf kan ergee tureen Waaqayyo Gooftaa (Adonaay Yaahiwee) Kakuu Haaraatti Hafuura Qulqulluu ta'uusaa nutti dubbatameera. Kunis Hafuurri Qulqulluun Waaqayyo Yaahiwee ta'uusaa ragaa guddaa kan biroo ti.

Luqqisiiwwan Kakuu Moofaafi Kakuu Haaraa armaan olii waliin ilaaluun barreeffamoota dhiyeessineen dabalatee kakuuwwan lamaan walbira qabanii ilaaluu otuu hin feessisiin Waaqayyo Yaahiwwee ta'uusaa kan mirkaneessan luqqisiiwwan Kitaaba Qulqulluu bakka hedduutti argamu. Fakkeenyaf yoo jedhame luqqisii itti fufee haa ilaallu:

"Daawit ilmi Iseyi, nama isa Waaqayyo inni kan Yaaqoob ol ol qabee, dibee moosisee ture; inni faarfannaa qopheesee faarfachuudhaan Israa'elota in gammachiisa ture; dubbiin Daawit gara dhumaatti dubbate kanatti fufee jira. Inni, " Hafuurri Waaqayyoo ana keessa in dubbate, dubbiin isaa afaan koo keessa jira. Waaqayyo inni kan Israa'el dubbateera; inni kattaa Israa'el ta'e anaan, 'Namni qajeeltootti namoota bulchu, inni sodaa Waaqayyootiin in bulcha. Inni akka ifa ganamaa ti, akka biiftuu yeroo duumessi hin jirre haleelee ifuu ti, akka ifa aduu isa erga bokkaan roobee caamee lafa keessaa marga biqilchuutis' jedhe" (2 Sam. 23:1-4).

Kutaa kanatti mootichi Daawit Hafuura Waaqayyoofi Waaqayyon walqixxeessee ilaaluun amala Waaqummaa tokkoon yoo dubbatu argina.  Kakuu Haaraa keessaa fakkeenya tokko kan dhumaa haa fudhannu:

"Phexros yommus, "Anaaniyaas, akka ati hafuura qulluluu sobduuf , horii araddaa kee itti gurgurte keessaas akka dhokfattuuf, maaliif Seexanni yaada garaa kee keessa guute? Qabeenyi kee kun utuu ati hin gurguriin kan kee ture, erga gurgutees horichi harka kee keessa jira mitii ree? Attamitti garaa kee keessatti kana malte ree? Ati namoota sobde miti, Waaqayyyoon sobde malee" jedheen" (HoE. 5:3-4).

Bakka kanatti Ergamtichi Phexros Hafuura Qulqulluu sobuun Waaqayyon sobuu akka ta'e dubbachuun isaa Hafuurri Qulqulluun Waaqayyo ta'uusaatti akka amane argisiisa.

Kitaaba Qulqulluu keessatti Hafuurri Qulqulluun amala Waaqummaa guutummaan of keessaa qabachuusaa ibsameera. Fakkeenyaf yoo jedhame:

Uumaadha – Job. 33:4, Phar. 33:6, 104:29-30, Job 33:4Miluu'i Bekulaahee (bakka hundumaa kan argamuudha) – Far. 139:7-10, 1 Cor. 2:10-11Aaa'immaare Kulu (Hundumaa kan beekudha) -Isa. 40:13, Joh. 14:16, 1 Cor. 2:10-11Kehaalii kulu (hundumaa kan danda'u) – Mik. 2:7, Phar. 104:30, Matt. 12:28, Rom. 15:18-19.Kan bara baraa ti – Isa. 48:16, Isa. 61:1, Joh. 14:16, Heb. 9:14.

Hafuurri Qulqulluun qaama eenyummaa qabaatu (ana ofin jechuu kan danda'u) ta'uusaa fi warreen tokko tokko akkuma jedhan humna qofa ta'uu dhiisuu isaa Caffaanni Qulqulluun ragaa ba'a. Fakkeenyaf yoo jedhame:

Humni eenyummaa hin haawanne waantota armaan oliin ibsaman raawwachuu hin danda'u.
