# Seensa

Qajeelchi qo’annaa kun kan qophaa’e amantaan Islaamummaa maal akka barsiisu fi ilaalcha hubannoo addunyaa (world view) Musliimotaa Kiristiyaanonni akka beekanii fi hubatan gargaarufi.

Kuni barreeffamoota walduraa duuban ba’uudhaf karoorfaman keessaa kitaaba isa duraati. Brreeffamoonni biroo matadurreewwan biroo irratti kan xiyyeeffatan ta’u. Kuni garuu jaalala Kiristos Musliimotaaf qooduu, Islaamummaa irraa warra deebi’aniif kunuunsa gochuu fi akkasumas barsiisa Islaamummaa barsiisa Kiristiyaanummaa waliin dnetomsiisuu irratti xiyyeeffata. Kitaabonni biroo hundinuu akka agarsiisanitti Islaamummaa fi Kiristiyaanummaan baay’ee garagaradha. Haata’u malee ummatoonni Kiristiyaanas ta’anii Musliimonni gaarummaa fi gara laafummaa nikabaju; akkaasumas nagahaan waliin jiraachuu nidanda’u. kanas eddoo tokko tokkotti baroota dheeeradhaaf godhaanii jiru. Kirisitiyaanonni Musliimota yeroo hundaa jaalala Kiristosin jaalachuu qabu.

Kaayyon qajeelcha qo’annaa kanaa Kiristiyaanonni Islaamummaan Kiristiyaanummaa irraa hangam adda akka ta’e akkasumas kana dura Islaamummaan addunyaa irratti haala akkamiitin akka babal’ateefi ammas akkamitti akka babal’atu akka hubataniifidha.Boqonnaan inni dhumaa kan irratti xiyyeeffatu Wangeela Musliimotaaf qooduu irrattidha. Dhuma irrattis jechoota Islaamummaa tokko tokko hiikuu kan danda’u hiikni jechootaa ni argama. Boqonnoota hunda keessattis kutaalen Macaafa Qulqulluu dabalamanii jiru. Kunneenis boqonnaa tokkoon tokkoon irratti ilaalchaa Kiristiyaanummaa akka kennaniifidha.

Qajeelfamichi namoota dhuunfatiif qa’annaa dhuunfatiif yookan immoo dursitoota waldaa Kiristiyaanatiif gareewwan tajajiltootaa leenjisuuf faayidaa irra ooluu nidanda’a. Kunis walitti aansudhaan kutaalee yeroo kudha sadi’iitin yookan koonfiransii leenjitiin ta’uu nidanda’a. Boqonnaan qajeelcha qo’annaa kanaa tokkoon tokkoon isaa mataa isaatin guutuu ykn ofdanda’a dha. Kanaafuu boqonnaa hundaa qo’achuudhaaf yeroon hinga’u taanan geggeessan garee garichaaf baay’ee barbaachisaa akka ta’e kutaa itti amanu filaachuudhan dursa kennuu nidanda’a.

Dhuma boqonnaa tokkoon tokkoon irratti eddoowwan duwwan deebin irratti barreeffamuu danda’an gaafilee waliin kennamanii jiru. Qajeelchi qo’annaa kun garee leenjiisuudhaaf kan ooluu yoo ta’e dura bu’aan garichaa gaafilee kanneen fayyadamuudhan marii fi falmii gaggeessuu  ni danda’a. Gafileen kunneen yaada qofadha. Dura bu’aan tokko gaafilee tokko tokko keessaa hanbisuu fi  itti dabaluu nidanda’a.

Meeshichi qo’annaa kun qo’annaa torbeetiif kan oolu yoo ta’e immoo dura bu’oonni qo’annaa torbee haala kanatti aansee jiruun qindeessuu ni danda’u:-

walga’icha kadhannaadhan banuu
Boqonnaa torbee darbee keessa deebi’uu
mata duree torbee kanaa irratti barumsa laachuu
gaafilee fayyadamuudhan marii garee gaggeessuu
kadhannaa fi waaqeffannaa jijjabeessuu
Kutaa barnootaa kamiyyuu dura qophiidhaf yeroo kennuu dura bu’aan tokkoo baay’ee kan isa fayyadu ta’ee ni argata. Qajeelchicha sirritti qo’achuuun akkasumas eegumsaa fi gaggeessa gooftatiif kadhaachuun barbachiisaadha. Qajeelchi kun hamma tokko kan ilaallatu wantoota Musliimonni godhaanii fi amananiidhaa. Haata’u malee kuni ibsa bu’uuraa qofadha. Waa’ee Islaamummaa itti dabalanii warra barachuu fedhaniif qayyabannoo dablataa fi yaadoonni tokkoo tokkoo armaangaditti kennamanii jiru.
