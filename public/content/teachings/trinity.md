# Sadan-tokko/Sillaasee

Waaqayyo bifa amalaa fi aboo isaa addunyaa uumame keessatti mul'iseera. Uumamni dinqisiisaa ta'e waantonni isa keessa jiran hin beekamne, seerri, bareedinnii fi guddinni isaa waa'ee Gooftummaa Waaqayyoo wanna nutti himu in jiraata. Garuu keessumaa Caaffata Qulqulluu keessatti waa'ee isaa nutti himeera. Waaqayyo haala addaa ittiin mataasaa Kitaaba Qulqulluu keessatti mul'ise keessaa tokko Sadan-tokkummaa (Sillaasee) ta'uu isaa ti. Waldaan Kiristiyaanaa qaamota sadan Waaqayyoo "Sadan-tokko/Sillaasee" jechuun waamti. Kana qalbifachuuf yaada nama rakkisu ta'uusaa haalamuu yoo baateyyuu Waaqayyon hubachuuf murteessaadha. Karaa qaamota Sadaniin mataa isaa akkamiin akka ibse qorachuun  caalmaatti Waaqayyoon hubachuu dandeenya.

Kes. 6:4 "Yaa Israa'el dhaga'i! Waaqayyo gooftaan keenya Waaqayyo isa tokkicha!" jechuudhaan dubbata. Kitaabni Qulqulluun Waaqayyo tokkichi jiraachuusaa ifatti dubbata. Garuu amalaan tokko fi kan bara baraa ta'uusaa akkanumas walqixa kan ta'an qaamota sadaniin jiraata. Kitaabni Qulqulluun qaamota sadan Waaqayyo Abbaa, Ilmaa fi Hafuura Qulqulluu jechuun walqixxummaan isaan kaa'a (Mat. 28:19; 1Phex. 1:2, 1Qor. 12:4-6, 2Qor. 13:14). Qaama yemmuu jennu eenyummaa jechuu keenya (person) malee qaama ijaan arginu jechuu akka hin taane hubachuun barbaachisaadha.

Sadan-tokko nuuf iccitii guddaadha. Isa kana akka ta'uu malutti ibsuu kan danda'u fakkeenya kamiyyuu waan hin qabaanneef Waaqayyo tokko ta'uun qaamota sadan keessa akkamiin akka jiraatu guutummaatti hubachuu hin dandeenyu. Hubannoo keenya daangeeffamaa ta'een Waaqayyo keenya guutummaatti hubachuun kan hin danda'amneedha (Isa. 55:8-9). Garuu Kitaabni Qulqulluun kana barsiisuusaatiin ibsa itti kennuufi barsiisuu dandeenya. Elektrikiin jiraachuusaa amanuu fi itti fayyadamuuf dirqama hojimaata elektirikii beekuun akka nurra hin jiraanne akkuma kana ammoo waa'eeSadan-tokkooguutummaatti hubachuudhaa yoo baanneyyuu jiraachuun isaa garuu dhugaadha.

Bishaan akka fakkeenyaatti hubadhaa. Bishaan wanna tokkodha (H2O). Garuu dhangala'aa, Gaazii (hurkaa) fi cabbii ta'uun bifa sadiin argama. Waaqayyos maalummaa tokko kan qabaatu ta'uun qaama sadiin mul'ateera. Garuu haala bishaan irraa adda ta'een qaamonni Sadan-tokkoo karaa dhugaa ta'een walii isaaniin tokkummaa godhatu. Dubbii gabaabaan yoo ibsamu, Waaqayyo akka Sadan-tokkoo isaatti hojii sadii hojjeta. Waaqayyo Abbaan kan karoorsu, Waaqayyo Ilmaa karoora kan raawwatuu fi Waaqayyo Hafuura Qulqulluun ammoo karoorri isaa jireenya amantootaa keessatti akka hojjetu kan godhuudha. Tokkummaan Sadan-tokkoo keessa jiru ilmi Waaqayyo Abbaaf bitamuusaa fi Hafuurri Qulqulluun Waaqayyo Abbaa fi Ilmaaf bitamuusaa in ibsama. Waaqayyo Abbaan ammoo karoora Waaqummaasaa galmaan akka isaaf ga'uuf aboo hundumaa Ilma isaatiif kenneera.

## Waaqayyo Abbaa

Waaqayyo Abbaan Sillaaseef qaama jalqabaa yoo ta'u uumama uumuu fi namoota hundumaaf walqixa kan ta'e gaarummaasaa kennuun warreen itti amananis ta'e kanneen hin amanneef abbummaa isaa argisiiseera. Kitaabni Qulqulluun akkuma dubbatu namoota qajeelotaa fi qajeelota hin taaneef aduu in baasa, Qulqulluutaa fi cubbamootaaf bokkaa in roobsa. Bineensota manaafi daggalaa kan soorudha. Warreen mootummaa isaa keessa jiraniifis ta'e isaatiin mormaniif otuu hin hafiin wanna isaan feessisu kennaaf (Mat. 5;45, 6:25-33, 7:9-11,  Luq. 6:35, Ibr. 12).

Waaqayyo Abbaan hariiroo adda ta'e amantoota isaa waliin qabaachuusaa ifa godheera. Waaqayyo ayyaanasaatiin gara ofii isaatti nu harkisuun ijoolleesaa godhatee gara maatii isaatti nu dabaleera (Yoh. 6;44, 1:12, 1Yoh. 3:1). Waaqayyo "Abbaa Abbaa" jennee akka isa waammannuuf hafuura Ilma isaa gara laphee keenyaa ergeera (Rom. 8:15-16, Gal. 4;6-7). Ijoollee Waaqayyo isa jiraataa jedhamanii waamamuufi Ilma isaa Kristos waliin dhaaltuu mootummaa isaa jedhamanii waamamuun kan nuuf maluun caalumatti iccitii guddaadha (Rom. 8:17).

Waaqayyo Abbaan ijoollee isaa godhatee gara maatii isaa nu dabaluun faayidaafi itti gaafatamummaa of keessaa qaba. Efe 1:3-14 irratti akkuma barreeffame, Waaqayyo Abbaan akkasumaan tola dhiifama nuu godheera waantota gaarii hundumaa ammoo nuuf kenna. Qulqulluu fi mudaa kan hin qabaanne taanee akka jiraannuuf nuuf kaayyoo addaa qaba. Karoora isaa ammoo Kitaaba Qulqulluu keessatti nuu ibseera. Abbaa mudaa hin qabaanne ta'ee ijoollee isaa hanga taanee jirrutti karaa hafuura isaatiin fayyadamuun jaalala isaa nuu mirkaneesseera. Ijoollee jaalatamoo gammachuu barabaraa fi wabii qabaatanii karaa amantiin gara Waaqayyo Abbaa akka dhufnu ta'eera (Efe. 3:12).

Waaqayyo Abbaan abboommii isaa raawwachuufi biyya lafaa isaa waliin araarsuuf bakka bu'aa isaa ta'uunii fi kkf itti gaafatamummaa ijoollee isaaf kenneera (Yoh. 14;21, 2Qor. 5:20). Akka miseensa maatii isaatti miseensa Waldaa Kiristiyaanaa ta'uu, waaqeffachuunii fi galateeffachuun isaaf deebii kennuun nurraa eegama (Ibr. 10:24-25, 1Phex. 2;9).

Waaqayyo eenyu ta'uusaa yoo qorannu amalasaa baraa deemna. Waaqayyo akkumaSadan-tokkooisaatti sammuu keenyaan olii fi karaa Waaqayyo Abbaa ammoo Abbaa keenya isa dhiyeenyaa ta'uusaa Caaffata Qulqulluurraa baranna. Yaada keenya guutuudhaan Gooftaafi Abbaa keenya jennee isa waamuu dandeenya!

## Xummura

Waaqayyo sagalee isaa keessatti waa'ee mataa isaatiif kan dubbate beekuun barbaachisaadha.Waaqayyo qaama sadaniin kan jiraatu Waaqayyo tokkicha (Sillaasee)dha.Waa'eeSadan-tokkooguutummaatti hubachuun kan hin danda'amne ta'us akka amannuuf Kitaabni Qulqulluun nu barsiisa.Waaqayyo Abbaan, Waaqayyo Ilmaa fi Waaqayyo Hafuurri Qulqulluu ga'ee hojii kan mataasaanii ni qabaatu.Waaqayyo Abbaan Sillaaseef qaama kan jalqabaa ti.Ijoollee isaa hanga taanee jirrutti itti gaafatamummaa fi faayidaawwan adda ta'an in qabaanna.
