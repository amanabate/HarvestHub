# Boqonnaa 1 – Humnaa fi Jaalala Waaqayyoo

Waaqni Keenya Jabaadha!

 Macaafa faarfannaa 33.6-11 dubbifadhaa .

Waaqayyo samii fi lafa akkasumas wanta isaan keessa jiru kamiyyuu uumee jira. Saayintistoonni kan dubbatan hawaan nuti beeknu waggaa ifaa biliyoona baay’eedhaan lakkaa’amu akka fagaatudha. Waggaan ifaa waggaa tokko keessatti ifti fageenya hammamii akka adeemuu kan mul’isuudha. Kunis saffisa ifaa yoo xiinxaltan baay’ee fagoodha; sakandii tokko keessatti 300,000km jechuudha. Hawaan gaalaaksota biliyoonan lakkaa’aman yoo qabaatu gaalaaksonnis immoo urjiiwwan biliyoonan lakkaa’aman qabu. Macaafa Seera uumamaa boqonnaa tokko irratti Waaqayyo afaan isaatii jecha isaa qofa akka dubbatee fi hawaan guddaan kun akka uumame ni dubbata.

Waaqayyo jabadha. Inni wantoota nuti arginuu fi kan nuti hin argine ni to’ata. Inni humnoota, mootummoota, pirezidaantota, bulchinsotaa fi gaggeessitoota addunyaa ni too’ata. Humni isaa baay’ee guddaadha. Gooftaa Yesusis du’aa kaasee jira.

Waaqayyoon Waaqni Keenya Dhuunfatti Kan Baramuudha

Efesoon 1.3-5 dubbifadhaa.

Guddinni hawaa Islaamummaa waliin walitti dhuufeenya maalit qaba jettanii dinqisiifattu ta’a. Gaaridhaa; Islaamummaa irratti barumsa walittifufaa kana jalqabuu keenyan duratti kan mirkaneeffachuu qabnu Waaqni keenya eenyu akka ta’eedha. Waaqni Musliimonni waaqeeffatan baay’ee jabaadha; garuu jiraataa miti. Musliimonni kan amanan Waaqayyo baay’ee fagoo akka ta’ee fi dhuunfatti beekkamuu akka hin dandeenyedha. Kanaafuu isa waliin dhuunfatti hariiroo uumun hin danda’amu.Waaqayyoon keenya guddaa fi jabaa akkasumas wantoota raawwachuu kan danda’u qofa osoo hin ta’ini, inni dhuunfaa keenyattis Waaqadha. Inni tokkoon tokkoon keenya fi waa’ee keenyas wanta hundaa nibeeka. Nuufis kunuunsa nigodha. Nujaalata; kanaafis nuwaliin dhuunfatti hariiroo gochuu fedha. Inni Waaqa humnaa fi jaalalaati. Innis ilmi isaa fannoo irratti du’a baay’ee sukaneessaa ta’e akka du’u erge. Kunis karaa isaatin cubbuun kiyyaaf kan keessan akka dhiifamuufi. Inni kana mara nu jaalatee jira.  Ilmi isaas fannoo irratti kan du’e nuti garbota isaa akka taanufii miti. Inni kan du’e cubbuu fi du’a irraa bilisa nubaasufi. Itti aanses nuti ijoollota isaa kan dubaraaf kan dhiiraa akka taanufi. Samiifi lafti osoo hin uummamin durayyuu inni nuuf yaaduun isaa wanta baay’ee nama dinqisisuudha. Nuti ijoollee Waaqayyoo isa hawaa baal’aa kana jecha isaa qofaan uumeti. Akka ijoollee Waaqayyootti humni Waaqayyoo guddaan karaa Kiristosin nukeesstti nihojjeta. Filiphisiyuus 4:13 irratti sagaleen goofta “jabina gooftan anaaf kennuun waan hundumaa goochuu nan danda’a” jechuudhan ni dubbata.

Garaagarummaa Islaamummaa fi Musliimota Gidduu Jiru

2Qorontos 4:1-6 dubbifadhaa

Barnoota Islaamummaa irratti qopha’e kanaan waa’ee Islaamummaa wantoota heddu nibarannaa. Wantoonni barattan tokkoo tokko keessa keessan jeequu, abdii kutaachu, aarii yookin immoo sodaa isinitti fiduu ni danda’u ta’a. Qo’annoo Islaamaa kana jalqabuu keenyaan duratti amantaa Islaamaa fi Muslimoota amantaa kana hordofan gidduu  garaagarummaa jiru hubachuun barbaachiisadhaa. Addunyaa irratti isaan qofti amantaa dhugaa akka ta’an kan dubbatan amantaalen heddu ni jiru. Islaamummaanis isaan keessaa isa tokkoodha. Haata’u malee Islaamummaan Yesus Kiritos Waaqa ta’uu isaa fi cubbuu keenya irraa nu fayyisuuf nuuf du’uu isaa ni gana. Macaafni Qulqulluun garuu gara Waaqayyo Abbaa kan nu fiduu danda’u karaan inni dhugaa Yesus qofa akka ta’e Yohannis 14:6 irratti ni dubbata. Kana jechunis akka Kiristiyaanotaatti kan nuti amannu karaan fayyinaa tookko qofa akka ta’ee fi innis Yesus Kiristos akka ta’eedha. Kanaafuu amantaan Yesus Waaqaa miti jedhu kamiyyuu amantaa sobaati.

2Qorontoos 4:1-6 irratti Macaafni Qulqulluun kan dubbatu akka Kiristiyaanota ta’uu keenyatti dhugaa labsuu fi kiyyoo amantaalee sobaa keessa namoota jiran gargaaruu akka qabnuudha. Qorontoos irratti kan dubbatu namoonni baay’en dhugaa misiraachoo Wangeela Yesus Kiristos akka hin agarre Seexanni akka isaan jaamsedha. Namoonni Seexanni jaamse kunneenis kan badan akka ta’an ni dubbata. Dubbii misiraachoo kana hin dhagahan taanan isattis hin amanan taanan bara baraaf ni du’u. Kanaaf nuti warra baduudhaf jiraniif Gooftummaa fi fayyisuu Kiristos labsuun akka nurra jiraatu nutti himamee jira. Kanaaf akka Kiristiyaanotaatti dubbii misiraachoo Kiristos Musliimota Kiyyoo Islaamaa keessa jiraniif labsuu qabna. Kaayyon kanaatis, baay’onni isa akka Gooftaafi fayyisaatti gara beekutti akka dhufaniifidha.

Islaamummaan dhugaa Macaafa Qulqulluu irratti amantaa hundeeffame akka hin taane kan beeknu ta’us nuti garuu Musliimota jaalachuutu nuuf ta’a. Akkuma namoota addunyaa irratti argamanii kan biroo Musliimonni namoota bifa Waaqayyootin uummamaniidha. Isaan garuu amantaa Islaamummaa keessatti kiyyeeffamanii jiru. Nageenya dhugaa dhabanii sodaa guddaa keessa jiraatu. Nagaha argachuufis erga du’anii booda eessa akka dhaqan mirkaneeffatanii hin beekan. Waaqayyo biyya lafaa baay’ee waan jaalateef warri isatti amanan hunduu jireenya bara baraa akka argataniif Ilmi Isaa Yesus Kiristos fannoo irratti akka du’u ergee jira. Nuti Waaqayyo akka Musliimota jaalatuu fi akka fayyanis fedha akka qabu ni beekna. Akka ijoollee Waaqayyoottis Musliimota dabalatee namoota hundattuu jaalala Waaqayyo agarsiisuu qabna. 2Qorontoos 5:14 irratti akkuma ilaallu Kiristos kan du’e namoota hundaafidha. Namoota biroof waa’ee isaa akka himnu kan nu taasisu jaalala isaa kanadha. Dubbiin Musliimota waliin taasisnu kamiyyuu jaalalaafi gara laafummaa irratti kan hundaa’e ta’uu qaba. Jibbi, lolli yookan sodaan walitti dhufeenya Musliimota waliin taasisnu keessatti gonkumaa eddoo hin qaban.

Jaalala Malee Sodaan Hin Jiru

Luqaas 12:4-12 dubbifadhaa.

Kiristiyaanonni tokko tokko Islaamummaa ilaalchisee sodaa guddaa qabu. Sodaa kana kan fidan sababoonni baay’en jiraachuu danda’u. Afrikaa fi addunyaa keessatti Islaamummaan haala sodaachisaadhan babal’achaa ni jira. Oduun addunyaa kan guutame dubbiiwwan Islaamummaa waliin wal qabataninidha. Fulbaana 11 gamoowwan Ameerikaa jiganii namoonni kuma heddu dhumuun isaanii, akkasumas waraanni Iraaqi fi Afgaanistaan xiyyeeffannoo oduuti. Islaamummaan baay’ee Mankaraareffamee Musliimonni baay’enis shororkeessummaatti amanaa jiru. Kiristiyaanotaaf namoota Musliimota hin taane biyyoota Musliimaa keessa jiraatan irra dhiibbaa guddaatu jira.

Waaqayyo baay’ee guddaa ta’uu isaa fi nutis ijoollee isaa ta’uu keenya yaadadhaa. Nuti ijoollee Waaqa guddaa kanaa waan taanef homaayyuu sodaachuu hin qabnu. Inni nu ilaala. Gooftan Yesus humnoota hamoofi wantoota namoonni gochuu danda’an gonkumaa akka hin sodaanne nutti himee jira. Inni baay’ee nu jaalata. Rifeensi mataa keenyatuu hammam akka ta’e ni beeka. Nuti wanta Waaqayyo ifa nuuf godhee fi wanta biroo kamiyyuu sodaachuu hin qabnu.

Akkasumas Inni kan nutti hime akkuma Inni nu jaalate nutis namoota biroo akka jaalannuudha. Abbaan keenya inni samii namoota biroo jaalachuu akka ajaja tokkootti nuuf kennee jira. Marqos 12:31 irratti ajajoonni gurguddoon lamaan Gooftan Yesus dubbate inni duraa Waaqayyo onnee, lubbuufi humna keenya hundaan jaalachuu yemmuu ta’u inni lammataa immoo olloota keenya akka ofii keenyatti jaalachuudha. Kutaa armaan olii keessatti ollaa jechuun nama lafa irra jiraatu kamiyyuu of keessatti hammata. Waaqayyo mootummaan isaa biyya lafaa irratti akka babal’atu kan fedhu jaalalaa fi dhiifamaani malee soda fi jibbaani miti.

Qophii hafuuraa

Efeson 6:10 dubbifadhaa.

Wanti nuti sodaannu tokkollee jiraachuu baatus Islaamummaa qo’achuu keenyan duratti qophii hafuuraa qabaachuu keenya mirkaneeffachuu qabna. Efeson 6:10-18 irraa wanti barachuu qabnu heddutu jira.

Inni duraa Waaqayyo jabaa fi humna qabeessa ta’uu isaa yaadannee nutis isarraa kan ka’e humna qabeeyyii ta’uu akka dandeenyu beekudha. Waaqayyo kan nutti himu nuti namoota kan ta’an Musliimota waliin wal loluu akka hin qabnedha. Yaadachuun kan nurra jiraatu Musliimota jaalachuufi isaanif kadhachuu malee isaanin loluu miti. Waaqayyo kan nutti hime lolli keenya lola hafuuraa hafuurota hamoo ta’an waliin kan taasifamu akka ta’eedha.

Humnoonni hafuuraa dhugumaan Islaamummaa keessa ni jiru. Kanaafuu eegumsa Gooftaa keessa jiraachuu keenya mirkaneeffachuu qabna. Humnoota hafuuraa kanneen fuuldura jabina cimaadhan akka dhaabbachuu qabnu nutti himamee jira. Kunis kan ta’u dhugaa fi jireenya qulqullaa’atiin yemmuu deddeebinuudha. Waaqayyo irratti amantii qabaachuu qabna. Amantiin kunis waraansa Seexanaa irraa nu eega. 1Yohannis 4:4 irratti inni nu keessa jiraatuWaaqayyoon, isa biyya lafaa keessa jiraatu Seexana irra humna baay’ee caalu akka qabu nutti himamee jira. Waaqayyo nu eega; sababni isaas inni nu keessa waan jiraatu fi nutis immoo ijoollee isaa waan taanefidha. Humnoota hafuuraa hamoota ta’an meeshaa ittiin waraannu nuuf kennee jira. Kunis sagalee Waaqayyooti. Nutis sagalee isaa sirritti yoo qo’anne Seexanaa fi akeeka isaa fuuldura humna guddaadhan dhaabbachuu ni dandeenya jechuudha.

Wanti dinqii ta’e inni gooftan nutti hime kan biroo immoo miilli keenya qophiidhan dhaabbachuu akka qabuudha. Kunis kan dhufu Wangeela nagahaa irraati. Nuti lola hafuuraa keessa kan jiraannu taanus wanti Gooftan nutti hime kaayyon lola kanaa maal akka ta’e yeroo hundaa yaadachuun akka nurra jiraatudha. Innis badiifi shororkeessummaa osoo hin ta’in jaalalaa fi nagadha.

Lola kana keessatti gaheen keenya oduu gammachiisaa Yesus Kiristos, jechuunis Wangeeela isaafi jaalala isaa babal’isuudha. Sababa kanaafis warri badii keessa jiran akka fayyina argataniifidha. Dhuma irrattis osoo wal irraa hin kutin kadhachuun nurraa eeggama. Yemmuu qo’annaa kana gaggeessinu Waaaqayyo eegumsa isaa akka nuuf baay’isu maqaa Gooftaa Yesusin kadhachuu qabna. Eegumsa, ija jabummaa, ogummaa fi amantii jabaan akka nuuf ta’u walii keenyaf kadhachuu nu barbaachisa. Warroota Islaamummaa jalatti gadadamaniifis kadhachuu qabna. Akkasumas warreen kiyyoo Islaamummaa keessa jiran kunneen Gooftaa Yesus gara amanuu akka dhufaniifi Abbaan keenya kan samii jabaa qofa osoo hin ta’in jaalalaan kan guutame akka ta’es akka hubatan kadhachuu qabna.
