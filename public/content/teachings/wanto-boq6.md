# BOQONNAA 6 – Bu'uurota Kiristiyaanummaa Marii'annoodhaf hin Dhiyaanne

Musliimota Afrikaa keessatti argamaniif karaalee deebii itti kennuu dandenyu heddutu jiru. Isaanis boqonnaalee muraasa itti aananii jiran keessati ni qo’anna. Deebiwwaan yemmuu kenninutti garuu bu’uurota amantaa keenyaa baay’ee barbaachisoo ta’an irratti ejjennoon keenya akka hin reeqqifne (akka oliif gadi hin jenne) mirkaaneeffachuu qabna.


Bara 390 dhalootaa Kiristoos booda dursitonni Kiristiyaanaa jiraatan amanatiiwwaan Kiristiyaaanotaa baay’ee barbachiisoo ta’an walittti qabanii barreessanii jiru. Kunis yeroo sana irraa jalqabanii Kiristiyaanonni bu’uurawwaan amantaa isaanii akka beekan isaan gargaaree jira. Ibsa amantaa ergaamootaas jedhamee waammaama.

Namoonni tokko tokko Kiristiyaanumaaa fi Islaamummaaan baay’ee akka walfaakkatan ni dubbatu. Eeyyeni; karaa tokko tokkoon walfakkatu ta’a. Nuti lachuu Waaqa tokkotti ni amanna. Macaafa Qulqulluu Kiristiyaanotaa keessatti akkasumas macaafota amantaa Islaamummaa, Qur’aanaa fi Hadisa keessatti seenaalee walfakkaaton ni argamu. Haata’uu maalee amantaalee kanneen gadi fageenyan yemmuu qorannu garaagarummaa gurguddoo arganna. Kiristiyaanonni Musliimota waliin walitti dhuufeenya uumuf yemmuu itti yaalanitti yeroo tokko tokko garaagarummaa kanneen tuquu dhiisuf niyaalu. Michuuwwan isaanii Musliimotaaf jaalattoota fi kabajdoota ta’uuf fedha qaban irraa kan ka’e bu’uuraalee Kiristiyaanummaa  tokko tokko irraattii ejjennoo laafaa ta’e qabatu. Amantaa Kiristiyaanaa keessatti waldaale fi manneen amanataa heddutu jiru. Haata’uu malee hundi keenyaa kutaalee qaama Kiristoos yemmuu taanu miseensa mana amantaa kamiyyuu yoo taane sagalee tokkoon kan itti waliigallu bu’uurota ni qabna.


Kutaalen Kiristiyaanaa baay’ee barbaachisoo ta’an tokko tokko kan mari’annaa keessa galchuu nuti hindandeenyedha. Sababa kamiinuu haata’u jijjiirramuu yookin dagatamuu hin qaban. Musliimonni Islaamummaa isaanii irratti cimoodh. Halumaa walfaakkatun Kiristiyaanummaa isaanitiin Kiristiyaanota cimoo ta’an ni kabaju. Qabxiilen armaan gadii akka Kiristiyaanummaa keenyatti kan itti ammannuu fi kan mari’annaa kessaa nuti hin galchine nuuf ibsu.




Kiristiyaanonni Kan Amannu

Nuti kiristiyaanonni Sadan Tokkummaa Waaqayyootti ni amanna. Afaan biraatin Waaqayyo qaamota sadi’iin kan jiru Waaqa tokko akka ta’e ni amanna jechuudha. Kunis Waaqayyo Abbaa, Ilmaa fi Hafuura Qulquluudha. Qaamoleen Kunneen Sadantokkoo (Trinity) walqixxeedha; akkasumas Waaqadha. Wanti kun hubachuuf baay’ee ulfaatadha. Ta’us amantiidhaan ni fudhaanna. Kana haala gaaridhaan kan nuuf deeggaran kutaalee Macaafa Qulqulluu keessaa Matewoos 28:19 fi 2Qorontoos 13:14 kan argaman yoo ta’u, kutaalen Kakuu Moofa fi Haarawaa heddu ni jiru.



Musliimonni Kan Amanan

Musliimonni Waaqni tokko qofa akka jiru ni amanu. Kunis kutaa amantaa isaanii baay’ee barbaachisaadha. Amantaa isaanii yemmuu labsan “Allaah malee Waaqni biraa hin jiru…’’ jedhu. Haata’uu malee Waaqayyo qaamota sadi’iin ta’uu akka danda’u hin amanan. Akkasitti yaadun isaanif baay’ee naasisaa fi cubbuudha. Musliimonni harki caalan Kiristiyaanonni waa’ee Sadantokkoo (Trinity) maal akka amanan yemmuu yaadan bitaa isaan gala. Sirummayyuu isaan keessaa tokko tokko Waaqayyo Abbaa, Maaryamiin haadha,Yesusimmoo Ilma Isaanii akka ta’e isaanitti fakkaata.


Dabalataanis Waaqayyo gonkumaa ‘‘Abbaa’’ jedhanii hin waaman. Allaahn isaanii baay’ee guddaa waan ta’eef namoonni gara isaatti dhihaachuu hin danda’an. Namni tokko Allaah waliin walitti dhuufeenyaa cimaa gochuu isaa irraa kan ka’e Abbaa jedhee waamuun kan yaadamu dhiisaatii, kan abjootamu hin ta’u.



Yesus Waaqadha


Kiristiyaanonni Kan Amannu

Akka kiristiyaanotaatti Yesus Kiristoos duuka ni buuna, Waaqas waan ta’eef isa ni waaqeffanaa. Inni qaama Sadantokko keessaa isa lammaffaadha. Yeroo hundaas Abbaa fi Hafuura Qulqulluu waliin akka ture ni amanna. Gara lafaa ni dhufe, Durbee Maaryamii irraas ni dhalate, akkasumas akka namaatti jireenyaa cubbuu hin qabne ni jiraate. Dhiifamuu cubbuu keenyaatiif jecha du’a fannoo nuuf du’e. Sana booddees du’aa ka’uudhan kara samiittii ol lixe; ilmaan namaa irraatti faraduudhaafis gara lafaatti deebi’ee ni dhufa.



Musliimonni Kan Amanan

Musliimonni Yesus Waaqa yookin ilma Waaqa jedhanii hin waaman. Isa waaqeffachuun Waaqayyoon akka arrabsuutti lakkaa’ama. Haata’uu malee Yesusin akka raajii guddaatti ni kabaju.Yesus jechuu irraa Issaa jedhanii isa waamu. Gara lafaa kan dhufe gara Muhaammad agarsiisuf qofa jedhu. Durbee Maaryamii irraa dhalachuu isaa irratti Kiristiyaanota waliin walii ni galu. Haata’uu malee fannoo irratti fannifamee du’uu isaa fi du’aa ka’uu isaa hin amanan. Musliimonni Issaan gara lafaa deebi’ee akka dhufuu fi Kiristiyaanota hunda Musliima akka taasisu; fannoowwan hunda akka caccabsu ni amanu.



Hafuura Qulqulluu


Kiristiyaanonni Kan Amannu

Hafuurri Qulqulluun qama Sadantokko keessaa isa sadaffa yemmuu ta’u, Waaqadha. Hafuurri Qulqulluun gara lafaatti kan ergameef Kiristoos yemmuu lafa irra jiru hojii jalqabe itti fufsiisufi. Kaayyon isaa nu gaggeessu fi gara Yesus agarsiisudha.


1Qorontoos 12 irratti akkuma nuuf barreeffame immookennaawwan hafuuraa nuuf kenna. Galaatiyaa 5 irratti akka barreeffametti yaada keenya fi jireenya keenyan caalatti Yesus akka fakkannu nu taasisa.



Musliimonni Kan Amanan

Qur’aanni si’a tokko tokko waa’ee ‘‘Qulqulluu’’ jedhee hafuura waamu tokko ni dubbata. Haata’u malee Musliimonni harki caalan kun kan agarsiisu ergamaa Gabri’eelii akka ta’ee yaadu. Musliimonni Muhaammaaditti kan itti mul’atee fi mul’atawwaan Qur’aana uuman kan kenneef ergamaa Gabri’eelii akka ta’e amanu.



Macaafni Qulqulluun Dubbii Waaqaayyooti


Kiristiyaanonni Kan Amannu

Macaafni qulqulluun namootaf kan dhufe duubbii Waaqayyooti. Waaqayyo Kakuu Moofas taa’ee Kakuu Haarawa barreessudhaaf barreessitoota namaa heddu fayyadamee jira. haata’u malee hundi isaanituu yemmuu barressan Hafuura Waaqayyootin gaggeeffamanii turan. Macaafni Qulqulluun kan hin geeddaramne jedhamee jira. Tokkoon tokkoon kutaa Macaafa Qulqulluu hundi dhugaadha jechuudha. Dogoongora maleedhas. Wanti tokko itti dabalamuus ta’ee irraa hir’ifamuu hin danda’u.



Musliimonni Kan Amanan

Musliimonni Allahn Toohra (kitaabota Macaafa Qulqulluu shanan isaan duraa). Faatrfaannaa fi kakuu harawaa kennee jira jedhanii amanu. Kitaabilee qulqullonni isaan jalqabaa kanneenidha. Haata’u malee Yihuudonnii fi Kiristiyaanonni itti yaadanii akka Macaafa Qulqulluu jijjiiranii fi sababa kanaanis Macaafni Qulqulluun gonkumaa amanamuu akka hin qabne yaadu.


Musliimonni Qur’aanii isaanii baay’ee qulquluu ta’ee ilmaan namaatif Allaah biraa kan ergame ajajaa fi qajeelcha isa dhumaa akka ta’e amanu. Qur’aanni sagalee Waaqayyoo karaa ergaamaa Gabri’eelitiin darbee yaadannoo samuu Muhaammaad irraatti kan barreeffameedha jedhanii amanu. Qur’aannis Macaafa Qulquluu akka bakka bu’e ni amanu.



Fayyina


Kiristiyaanonni kan amannu

Faayyina jechuun cubbuu keenya irraa kan ka’e adabbii bara baraa nurra ga’u irraa hafuu jechuudha.Yemmuu lubbuun keenyaa dabartu gara motummaa Waaqqayyoo ni adeemna. Waaqayyo wanta nu jaalateef tokkoon keenya iyyuu kanaaf ga’umsa kan hin qabne yoo taanellee, kenna fayyinaa bilisaan nuuf kenne. Kunis ‘‘ayyaanan fayyuu’’ jedhameet beekkama. Kun kan danda’amus qaama Sadantokko keessaa isa lammataa kan ta’ee fi dhiifamuu cubbuu keenyaatiif jecha kan du’e, karaa Yesus Kiristoosin qofa. Faayyinni akka kafaltiitti wanta kennamu waan hin taanef hojii hojjennee hin argannu.Yesus Kiristoositti ennaa amannu cubbuun keenya dhiifamee jira, jireenya bara baraatis argannee jirra.



Musliimonni Kan Amanan

Musliimonni Yesus Waaqa akka ta’e yookin cubbuu keenyaaf akka du’e hin amanaa. Allaahn isaan amanan abbaa dhiifamaa akka ta’ee fi cubbuu hundaa yeroo barbaade qofa akka dhiifama godhuu amanu. Musliimonni gara Jannataa akka galan Allaahn karaa akkamiituu hin qopheessine. Wanta gaarii hojjechuu fi seera Shari’aa kabajuudhaan karaa isaanitiin argachuuf caraaquu qabu. Lubbuun Musliima tokkoo yemmuu dabartu hojiin isaa gaarin yoo hojii isaa badaa caale gara jannataa qajeelumaan gala. Haata’u malee Musliimonniheddu gara jannataa galuu isaanii dura waa’ee hojii isaanii badaadhaf gara eddoo itti adabamanii ni deemu.



Waaqayyo Jaalala


Kiristiyaanonni kan amannu

Nuti Kiristiyaanonni kan amannu Waaqayyo kan nama jaallatu qofa utuu hin ta’in inni jaalala akka ta’eedha (Yoha. 4:8). Jaalalli isaa haala irratti kan hin hundeeffamne waan ta’eef jaalala isaa argachuuf wanta tokkoyyuu gochuun nurraa hin eeggamu. Macaafni Qulqulluun Waaqayyo addunyaa guutuu akka jaallatuu fi namni hunduu akka fayyu akka barbaadu nutti ni hima. Kunis wangeela Yohannis 3:16 irratti “namni isatti amanu akka jirenya bara baraa argatuuf malee akka in banneef Waaqayyoo akkasitti tokkicha ilma isaa hamma kennuufitti biyya lafaa jaallate” jedhamee duubbii ifa ta’een barreeffamee jira. Roomaa 5:8 ‘‘Waaqayyo garuu, utuma nuyi cubbamoota taanee jirruu Kiristoos nuuf d’uu isaatin hammam akka  nu jaallate in argisiisa’’


Kanaafuu cubbamoota yoo taneyyuu Waaqayyo nu jaalatee jira. Akkaa Waqummaa isaatti amala isaa kana ganuu hin danda’u.



Musliimonni Kan Amanan

Musliimonni dandeettin Allaah inni tokko jaalachuu akka taa’ee fi maqoota isaa 99 keessaa inni tokko ‘‘jaalataa” kan jedhu akka ta’e amanu. Haata’u malee jaalalli Allaah haalota irrratti kan hundeeffameedha. Allaah warra gaarifii qulqulloota malee cubbamoota hin jaalatu. Qur’aanni kutaa isaa isa kam irrattuu‘‘Allaahn jaalala’’ hin jedhu. Akka bu’uuratti Suufiizimii/Islaamummaa hafuurawaa irraa kan hafe jaalalli Islaamummaa keessatti eddoo xinnoo qaba. Suufiiwwan Allaah waliin walitti dhufeenya jaalala irratti hundaa’ee akka qabaatan ni carraaqu, ni fedhus.



Ilmaan Namootaa


Kiristiyyaanni Kan Amannu

Namoonni bifa Waaqayyootin uumamanii jiru. Hundi keenyaa ija Waaqayyoo fuulduratti wal qixa. Addaam ajaja Waaqayyoo waan cabseef cubbuun gara Addunyaa gale (Roomaa 5:12). Hundi keenyaa cubbamoota. Cubbuun Waaqayyo irraa adda nama baasa. Walitti dhufeenya Waaqayyo waliin qabnu haaromsuu fi dhiifama cuubbu argachuuf karaan ittiin dandeenyuhumna hojji Yesus Kiristoos fanno irratti hojjete amanuu qofaani.Waaqayyo waliin dhuunfaa keenyattii waliiti dhufeentaa gochuu ni dandeenya.



MusliimonniKan Amanan

Musliimonni bifa Waqayyootin akka uumamnee fi hundi keenyaa walqixa akka taane hin amanan. Islaamummaa keessatti warra Musliimota hin taane irra Musliimonni; dubartoota irra ammoo dhiironni caalmaa akka qaban amanu. Musliimonni namoonni cubbuu waliin akka dhalatan hin amanan. Hojii gaariii hojjechuudhan cubbuun akka haqamu ni amanu. Dabalataanis Musliimonni Allaah baay’ee qulquluu fi itti siquun kan hin danda’amne ta’uu isaa irraa kan ka’e ilmaan namootaa dhuunfaa isaanitti isa wajjiin walitti dhufeenya godhachuu akka danda’an hin amanan.


Ibsa Amantaa Ergamootaa

Nan amana hunduma kan danda’u Waaqayyo Abbaatti; uumaa waaqaf lafaatti. Tokkicha Ilma isaa Gooftaa keenya Yesus Kiristoositti; kan ulfeeffame, Hafuura Qulqulluudhan, kan dhalate Maryam Durbarraa, kan dhiphate, bara Philaaxos Phenxenichaatti, kan du’es kan awwaallames, Si’oolittis kan gadi bu’e, guyyaa sadaffaatti du’aa kan ka’e, gara waaqattis kan ol ba’e, gara mirga Waaqayyo Abbaas kan taa’e, achiis kan deebi’u, warra jiraniifi warra du’anitti kan faradu. Nan amana Hafuura Qulqulluutti; tokkicha qulqullaa’aa waldaa Kiristiayaanatti; tokkummaa qulqullootatti, dhiifamuu cubbuus, du’aa ka’uu foonis, jireenya barabaraas, Amen.


2 Tasoloniiqee 2:15-17 dubbifadhaa

Macaafa Qulqulluu keessatti amantii keenyaan jabaannee akka dhaabbannu kutaalee nu yaadachiisan baay’een ni jiru; kunnenis Filiphiisiyuus 1:27-28 yaqoob 5:8 ni dabalatu Asuma 2tasolonqee 2:15-17 irratti Macaafni Quqlqulluun akka jabannee dhaabbanuu fi karaa sagalee Waaqayyoottin barsiisotaa nuuf darban jabeessinee akka qabnu nutty hima. Mormiiwwan cimmooyookin soda irra kan ka’e jeeqamnee wanta Macaafni Qulqulluun nu barsiise walitti mammakuu hin qabnu. Namoonni baay’ee Musliimota qaqqabuuf fedhii qabaatanii Musliimota akka hin mufachhiifne bu’uuraalee amanta Kiristiyaanaa tokko tokko kaasuu irraa of qusachuu darbees barbaachisoo akka hin taanee dubbachuu bni danda’u. kanas kan godhan gaarii gochuu fedhuu irraa kan ka’e ta’uu ni danda’a . Haata’u maalee  Macaafa qulqulluu irraa baranneef amanamoota ta’uu qabna. Hojjiiwwaan sirrii ta’an yemmuu hojjennuu fi sagalee Waaqayyoo isa sirrii yemmuu dubbanuWaaqayyoo Abbaafi Gooftaan keenya Yesus Kiristoos karaa humna hafuura Qulqulluutin jabinaa fi human nuuf kennaa.
